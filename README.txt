# GDENG2_GameEngine

Developer: Lanz Santiago
This is a requirement for DLSU GDENG-2 Course.

## How to Install

1. Clone this repository in an empty directory.
2. Execute Build.bat located in the directory
3. Open Engine.sln file to build the program (Assuming using VS).

The entry point (i.e. the main() function is located at Editor/Source/Main.cpp)
